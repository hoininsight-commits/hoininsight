# 한국은행 ECOS API 키 발급 가이드

## 📍 현재 위치: ECOS 메인 페이지

화면에서 보이는 메뉴:
- 통계검색
- 데이터 툴킷
- 통계공표일정
- 통계정보
- 주요통계제공
- 통계연구/간행물
- 고객지원

---

## 🔑 API 키 발급 방법

### 방법 1: 상단 메뉴 이용
1. **화면 오른쪽 상단**을 보세요
2. **"Open API"** 메뉴를 찾으세요
   - 또는 **"ECOS 오픈API"** 메뉴
3. 클릭!

### 방법 2: 직접 URL 이동
**이 URL로 바로 이동하세요:**
```
https://ecos.bok.or.kr/api/
```

---

## 📋 API 키 발급 페이지에서 할 일

### 1단계: 인증키 신청
- "인증키 신청" 또는 "API Key 발급" 버튼 클릭

### 2단계: 정보 입력
- 사용 목적: "개인 투자 분석용"
- 사용 기간: 1년 (자동 연장 가능)

### 3단계: 키 발급 완료
- 즉시 발급됨 (40자리 문자열)
- 예: `ABC123DEF456GHI789JKL012MNO345PQR678ST`

---

## 🔍 찾을 수 없을 때

### 옵션 A: 검색 사용
1. 페이지 상단 검색창에 "API" 입력
2. "Open API" 관련 결과 클릭

### 옵션 B: 고객지원 메뉴
1. 상단 메뉴에서 **"고객지원"** 클릭
2. "Open API" 또는 "인증키" 관련 메뉴 찾기

### 옵션 C: 직접 링크
**이 링크를 브라우저 주소창에 복사하세요:**
```
https://ecos.bok.or.kr/api/KeyStatisticList.do
```

---

## 📊 발급 후 확인할 것

### 인증키 정보
- 키 값 (40자)
- 발급일
- 만료일
- 일일 요청 한도: 10,000회
