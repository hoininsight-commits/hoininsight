# SCRIPT TEMPLATE (60-120s) v1.0

## 1. Hook (0~10s)
*   **목적**: 시선 강탈. "너만 이거 모르고 있다"는 뉘앙스.
*   **필수 요소**:
    *   질문형 오프닝 ("님들 ~ 알고 있었어?")
    *   긴박감 조성 ("딱 1분만 집중해 봐")

## 2. Observation (10~30s)
*   **목적**: 팩트 체크. Anchor 제시.
*   **필수 요소**:
    *   관측된 **숫자 2~3개** (지수 레벨, 계약 규모, 수익률 등)
    *   "표면적인 이유" 언급 (사람들이 아는 것)

## 3. Driver (30~70s)
*   **목적**: HOIN의 진짜 해석.
*   **필수 요소**:
    *   "하지만 진짜 이유는 ~다" (반전)
    *   Narrative Driver 설명 (자본 이동, 평가 기준 변화 등)
    *   전문 용어는 쉽게 풀어서 ("이게 무슨 뜻이냐면~")

## 4. Implication (70~95s)
*   **목적**: 그래서 시장이 어떻게 되는데?
*   **필수 요소**:
    *   관련 섹터/종목의 **파급 효과**
    *   월가/시장의 반응 ("재평가하기 시작했어")

## 5. Risk (95~110s)
*   **목적**: 균형 잡기 & 신뢰도 상승.
*   **필수 요소**:
    *   "물론 리스크는 있어"
    *   반대 시나리오 1개 제시 (금리 반등, 실적 쇼크 등)

## 6. Close (110~120s)
*   **목적**: Action & 구독 유도.
*   **필수 요소**:
    *   "우리가 기억해야 할 건 ~야" (한 줄 요약)
    *   "경제사냥꾼 구독과 좋아요" (CTA)
