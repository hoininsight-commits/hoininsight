# NARRATIVE TOPIC DEFINITION v1.1

## 1. 정의
Narrative Topic은 **"사람들이 체감하는 현상(Anchor)"**을 **"자본의 구조적 이유(Driver)"**로 설명하는, **영상 제작을 위한 단기 주제**이다.

## 2. 구조 (Anchor + Driver)
*   **Topic Anchor (겉)**: 뉴스 헤드라인, 급등락, 대형 계약. (사람들이 클릭하는 이유)
*   **Narrative Driver (속)**: 수급, 자본 이동, 평가 기준 변화. (HOIN이 설명하는 이유)

## 3. 원칙
1.  **Level 금지**: Narrative Topic에는 L2, L3, L4 같은 구조적 레벨을 부여하지 않는다.
2.  **Short-term**: 주로 3일~7일 정도의 유효 기간을 가진다.
3.  **No Recommendation**: 매수/매도 추천이 아니며, 현상의 배경 설명에 집중한다.

## 4. 목적
*   **영상 제작**: 60초~120초 분량의 쇼츠/릴스용 스크립트 소스.
*   **Traffic Hook**: 대중적인 관심을 Structural Insight로 연결하는 관문.
