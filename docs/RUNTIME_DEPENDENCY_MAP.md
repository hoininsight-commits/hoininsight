**Purpose**: Identification of active runtime paths and legacy artifacts for safe ecosystem slimming.

## 1. Analysis Overview
This map was generated by tracing imports, file operations (open/read/write), and fetch calls across core engine and dashboard entry points.

## 2. Core Entry Points
- **Engine**: `src/ops/run_daily_pipeline.py`, `src/engine/__init__.py`

## 4. Special Verification results
- **data/facts/**: **Runtime Critical**. Actively used by `FactFirstInputLoader` and several Fact Connectors.
- **docs/data/**: **Runtime Critical**. The dashboard fetches these directly (especially `manifest.json`).
- **data_outputs/**: **Historical / Archive**. Confirmed as a legacy mirror in `run_publish_ui_decision_assets.py`.
- **src/legacy/**: **Runtime Critical**. Many core engine functions still import from this directory.
- **archive/**: **Historical / Archive**. Used as an output destination for backups, but not as a required input for today's runtime.
