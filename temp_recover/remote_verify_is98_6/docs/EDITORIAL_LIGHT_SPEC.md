# EDITORIAL_LIGHT_SPEC.md

본 문서는 고품질 에디토리얼(TRUST_LOCKED) 공백 시 운영 리듬을 유지하기 위해 발행되는 '저강도 구조 해설형 콘텐츠'인 `EDITORIAL_LIGHT` 레이어의 규격을 정의합니다.

## 1. 운영 및 트리거 규칙

- **트리거 조건**:
  - 최근 3일간 `TRUST_LOCKED` 또는 `EDITORIAL_CANDIDATE` 발생 건수가 0일 때.
  - 현재 `MacroState`가 `Neutral`이 아닐 때 (시장에 읽을만한 구조적 변화가 있을 때).
- **상태 정의**: `EDITORIAL_LIGHT` (관찰 콘텐츠)
- **노출 우선순위**: `HOLD` 토픽들보다 상단에 위치하되, `TRUST_LOCKED` 보다는 하위.

## 2. 콘텐츠 생성 가이드라인 (헌법 준수)

| 항목 | 규정 |
| :--- | :--- |
| **Actor** | 매크로 주체(Actor) 또는 섹터 주체까지만 허용 (기업명 금지) |
| **Ticker** | ❌ **절대 도출 금지**. (실데이터가 부족한 상태이므로 추측 방지) |
| **Why-Now** | "강제/적기" 표현 대신 "구조적 에너지가 축적되고 있는 이유"로 완화. |
| **주의 문구** | "본 콘텐츠는 종목/행동 지시가 아닌 구조 해설입니다." 상시 노출. |

## 3. 스크립트 구조 (5단계 고정)

1. **정의 (Definition)**: 반복 감지되는 구조적 흐름 (관찰 보고).
2. **표면 (Surface)**: 일반적인 관측 포인트 요약.
3. **오해 (Misread)**: 뉴스/차트에 아직 반영되지 않은 이유 (잠복기 구조).
4. **구조 강제 (Force)**: 현재 에너지가 축적 중이며, 트리거 발생 시 증폭 가능성 언급 (단정 금지).
5. **결론 (Conclusion)**: "지금은 결론이 아닌 구조를 기억해야 할 시점"으로 마무리.
